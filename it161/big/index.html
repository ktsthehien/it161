Welcome

This is my BIG website. In this website, I have applied the fundamental mechanics of web development tools and resources that I learned from class IT161.

In these tabs, I have presented my research on important web development topics such as: Social Media Optimization (SMO), Accessibility, and Web Forms. The website also offers a responsive navigation menu with dropdown, Google calendar, Google map, and Gallery. In this project, I applied some web development skills in writing basic HTML & CSS code, using industry standard web directory structure and file naming conventions, constructing basic responsive websites, and creating contact form and applying reCAPTCHA. Besides, I have applied the basic mechanism of web development tools and resources such as Github, Brackets, FileZilla, website inspector to build this website.

To develop and maintain my web knowledge and skills, I have made a habit of using HTML, CSS, and JavaScript regularly, researching new trends and tools in the field, and developing teamwork skills.
